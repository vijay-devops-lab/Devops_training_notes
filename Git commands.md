
- To initilize a git repo 

```
	git init
```

-  To get the status 

```
	git status
```

-  To move source code from working  to staging area 

```
	git add .
	// if u what to add all file of folder 
	git add text.txt 
	// if you want to add specific file
```

-  To moving the source code from staging to local state 

```
	git commit -m "message"
```

-  to see the past commit 

```
	git log
```

-   **branch is** a lightweight, movable pointer to a specific commit

```
	git branch 
```

 - to create new branch  

```
	 git branch dev
```

-  to move to the branch

```
	git checkout dev
```

-  to merge 2 branch dev<-dev2

```
	git checkout dev
	git merge dev2
```

-  to resolve conflict 

```
git mergetool
```
